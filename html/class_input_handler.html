Detailed Description

The InputHandler class manages the user input and game logic.

The InputHandler class is responsible for handling the userâ€™s keyboard inputs, managing the game state (score, lives, difficulty), updating the UI, and managing the background music during the game. It provides three difficulty levels (Easy, Medium, Hard) and incorporates functionality to pause, resume, and restart the game.
